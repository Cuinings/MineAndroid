# Tools (开发辅助工具)

项目开发过程中使用的辅助脚本和工具集合。

## 工具列表

| 文件 | 用途 |
|------|------|
| `bundletool-all-1.18.3.jar` | Google Android App Bundle 命令行工具，用于构建、测试和部署 AAB |
| `generate_translations.py` | JSON → Android `strings.xml` 翻译文件生成器（中/英/俄/葡） |
| `init_translations.py` | 翻译文件初始化脚本 |
| `merge_translations.py` | 翻译文件合并脚本 |
