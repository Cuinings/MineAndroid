## StatefulGlowView 内发光 .9 图片支持（per-state 扩展）
- 新增属性 `sgv_glowDrawable`（reference，指向 .9.png drawable），设置在 attrs.xml StatefulGlowView styleable 中
- `drawInnerGlow()` 新增 .9 渲染分支：当 `glowDrawable != null` 时，跳过程序化 32 层 EVEN_ODD 环形发光，改用 NinePatchDrawable 铺满 View 并施加 `setTint(glowColor) + SRC_IN` 着色
- 新增公开 API `setGlowDrawable(Drawable?)`，传 null 恢复程序化发光
- .9 路径通过 `dispatchClipPath` 自动适配圆角裁剪；子类（AnimatedStatefulGlowView / FrostedAnimatedGlowView）无需改动即可继承
- **per-state 扩展**：`glowDrawable: Drawable?` → `StateDrawables` 五态数据类；新增 5 per-state attr (`sgv_glowDrawableNormal/Pressed/Focused/Selected/Disabled`) + `setGlowDrawableForState()` / `getGlowDrawableForState()` API；XML 解析支持 universal fallback + per-state override
