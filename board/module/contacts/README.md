# Contacts (通讯录)

智慧平板的通讯录 / 组织架构模块。

## 功能

- 组织架构展示与人员搜索
- 基于 ViewModel + Repository 架构
- 依赖 database 模块获取组织数据
- 使用 Lifecycle + Gson 进行数据管理

## 模块类型

Android Library（`com.cn.board.contacts`）

## 核心类

- `ContactsActivity` — 通讯录主界面
- `OrganizationViewModel` — 组织数据 ViewModel
- `OrganizationRepository` — 组织数据仓库
