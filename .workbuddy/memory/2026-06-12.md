# 2026-06-12

## board/proxy/test 模块：ParticipantBean 聚合 + CRUD

- 增强 `ParticipantBean`：新增 `mute` / `quiet` / `terminalAlias` / `aliasList` 字段，`id` 改为 `val`
- 新建 `ParticipantManager`：三源聚合管理器
  - `onStatusChanged(MTEntityStatus)` → 更新 online/mute/quiet
  - `onTerminalInfo(MtcConfTerminalInfo)` → 更新 terminalAlias/online（tId String→Int 容错转换）
  - `onEntityInfo(MTEntityInfo)` → 更新 name/aliasList
  - 批量：`onStatusBatch` / `onEntityInfoBatch` / `rebuildFromTerminalList`
  - 读取：`get(Id)` / `all` / `filter` / `findByUId` / `findByTId`
  - 删除：`remove(Id)` / `removeByUId` / `removeByTId` / `clear()`
- 核心理念：三路通知按 `Id(uId, tId)` 聚合，`LinkedHashMap` 保持插入顺序

## 线程安全 + RecyclerView Diff 改造

- `ParticipantManager` 加入 `ReentrantReadWriteLock`：
  - 写锁：`onStatusChanged` / `onTerminalInfo` / `onEntityInfo` / `remove` / `clear` 等所有变更方法
  - 读锁：`get` / `snapshot` / `all` / `size` / `isEmpty` / `filter` / `findByXxx`
  - 批量方法（`onStatusBatch` / `onEntityInfoBatch` / `rebuildFromTerminalList`）单次持写锁，减少锁争用
  - 新增 `snapshot()` → 返回不可变快照，直接交付 RecyclerView DiffUtil
- 新建 `ParticipantBeanDiff : DiffUtil.ItemCallback<ParticipantBean>()`
  - `areItemsTheSame` → 按 `Id` 判定
  - `areContentsTheSame` → 全字段比较，`@SuppressLint("DiffUtilEquals")`
  - `getChangePayload` → Bundle 携带变更字段 key，支持局部刷新
  - `companion object` 含 Payload key 常量 + `INSTANCE` 单例

## 角色排序 + aliasList 补充

- `ParticipantBean` 新增角色字段：`chairman` / `vip` / `speaker`（Boolean，默认 false）
- `ParticipantBean` 补回遗漏的 `aliasList: List<MtAlias>` 字段
- 新建 `ParticipantRole(uId, tId, chairman, vip, speaker)` 数据类，用于批量更新角色
- `ParticipantManager` 新增：
  - `onRoleChanged(role)` — 单成员角色更新（写锁）
  - `onRoleBatch(roles)` — 批量角色更新（单次写锁）
  - `onlineByRole()` — 在线成员按 chairman(3) → vip(2) → speaker(1) → 其他(0) 降序排列（读锁）
- `ParticipantBeanDiff` 同步更新：`areContentsTheSame` / `getChangePayload` 含 chairman/vip/speaker，companion 新增 KEY_CHAIRMAN / KEY_VIP / KEY_SPEAKER
